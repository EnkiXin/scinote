# `analysis_72b/` — Per-question note pairings (7B vs 72B noter)

One CSV per task joining the original question/options/gold with the three
predictions (Video-only / Video+7B-note / Video+72B-note) **plus** the actual
note text that the 7B and 72B noters generated for each video.

Generated by [`build_qa_note_pairs.py`](../build_qa_note_pairs.py).

## Columns

| Column | Source |
|---|---|
| `id` | Annotation id (e.g. `55531_clip_12_materials`) |
| `video_path` | Relative path inside the ExpVid HF dataset |
| `question` | The question text |
| `options` | JSON dict `{"A":..., "B":..., "C":..., "D":...}` (empty for non-MC tasks) |
| `gold` | Reference answer (JSON-serialized) |
| `pred_video`    | Prediction from **Video only** (`results_h200/qwen7b/`) |
| `pred_7B_note`  | Prediction from **Video + 7B-note**  (`results_h200_unified/c2/`) |
| `pred_72B_note` | Prediction from **Video + 72B-note** (`results_h200_unified_q72/c2/`) |
| `score_video` / `score_7B` / `score_72B` | Per-item score (0–1; F1 for seqgen/fitb, 0/1 for MC) |
| `note_7B`  | Full Stage-1 note generated by Qwen2.5-VL-7B  |
| `note_72B` | Full Stage-1 note generated by Qwen2.5-VL-72B |

## Quick analytical patterns

Easy filters to apply (e.g. in pandas):

```python
import pandas as pd
df = pd.read_csv("analysis_72b/qa_notes_materials.csv")

# Where 72B note FIXED an item that both Video and 7B-note got wrong:
df[(df.score_video == 0) & (df.score_7B == 0) & (df.score_72B == 1)]

# Where 72B note BROKE an item Video got right:
df[(df.score_video == 1) & (df.score_72B == 0)]

# Where 72B note ALSO won where 7B was already correct (no harm):
df[(df.score_7B == 1) & (df.score_72B == 1)]

# Sample 50 disagreements between 7B note and 72B note for qualitative reading:
df[df.score_7B != df.score_72B].sample(50)
```
